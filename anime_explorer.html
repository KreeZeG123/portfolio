AnimeExplorer

AnimeExplorer est une application web interactive permettant aux utilisateurs de tester leurs connaissances en animés de manière ludique. Développée dans le cadre de la troisième année de licence en informatique à l'Université du Mans, cette application utilise un jeu où les joueurs doivent deviner un animé en fonction de ses caractéristiques. À chaque tentative, des indices supplémentaires sur l'animé à deviner sont révélés, facilitant ainsi la progression dans le jeu.

Le projet a été réalisé en utilisant Angular pour le front-end, et Node.js avec une base de données MongoDB pour le back-end. Une API GraphQL permet d'accéder aux données des animés via l'API d'AniList. En plus du jeu, une fonctionnalité de découverte d'animés est incluse, permettant aux utilisateurs de trouver de nouveaux animés en fonction de critères spécifiques et d'une recherche floue par titre d'oeuvre. Un système de classement entre les joueurs ajoute une dimension compétitive à l'application.

Ce projet a été mené en collaboration avec d'autres membres de mon équipe. Le projet est disponible en open source sous la licence MIT, et il est possible de consulter le code de chaque auteur via leurs branches respectives.

Pour ce projet, nous avons développé une application web fonctionnelle permettant aux utilisateurs de tester leurs connaissances sur les animés à travers un jeu interactif. En plus de l'application, plusieurs documents ont été produits pour détailler les aspects techniques et méthodologiques du projet. Cela inclut une présentation du projet, une analyse du questionnaire utilisateur, ainsi qu'une description détaillée de l'API AniList utilisée pour interagir avec la base de données d'animés.
